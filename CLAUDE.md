# winformTester — E2E Testing Tool for .NET WinForms/WPF

Playwright 스타일의 E2E 테스트 프레임워크. .NET Framework 3.5 ~ 최신 .NET의 WinForms/WPF 앱을 지원하며 C#/VB로 테스트 작성 가능. 자동 실행 + 시나리오 재현 + 성공/실패 판정 + MP4 녹화.

## 하네스: E2E Tool Development

**목표:** .NET WinForms/WPF용 Playwright 스타일 E2E 테스트 툴을 설계·구현·배포.

**트리거:** E2E 툴 관련 작업(설계, 구현, 버그 수정, 기능 추가, 배포) 요청 시 `build-e2e-tool` 스킬을 사용하라. 단순 질문은 직접 응답 가능.
